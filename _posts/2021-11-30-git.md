### 创建repo后如何与本地项目相连接

进入本地项目目录并执行以下命令：
```
git init
git remote add origin ***.git
git branch -M master
git add .
git commit -m 'some commits'
git push -u origin master (push前可能需要pull -f 强制push)
```
